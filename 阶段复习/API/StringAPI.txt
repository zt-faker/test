字符串有哪些API可以使用？
1.char CharAt(int index)
	返回当前字符串中给定位置的字符串
2.int IndexOf(String str)
	返回给定字符串在当前字符串中位置，
	若当前字符串不包含给定字符串则返回-1.
3.int Length
	返回当前字符串的长度
4.static String.valueOf( )
	可以将其他类型转换为字符串
5.String trim( )
	去除字符串两边的空白字符
6.boolean toUpperCase( )
   boolean toLowerCase( )
	将字符串中放入英文部分转换为全大写或者全小写
7.String substring(int start ,int end)
	截取当前字符串中指定范围内的子字符串，
	两个参数为开始位置和结束位置的下标，含头不含尾
8.java.lang.StringBuilder
	用来修改字符串，其内部有个可变char[]，性能好，开销小
	append：添加字符
	replace：修改，替换字符
	delete：删除字符
	insert：插入字符
9.boolean startsWith（String str）
   Boolean endsWith（String str）
	判断字符串是否已给定字符开始或结尾
10.正则表达式
	boolean matches(string regex)
	用给定的正则表达式验证当前字符串的内容是否符合要求
	string[]split(String regex)
	将当前字符串按照满足正则表达式的部分进行拆分，
	将拆分后的字符串以数字形式返回
	String replaceAll(String regex,String str)
	将当前字符串中满足正则表达式的部分替换为给定字符串

+ ：  一个或者多个
*  ：0个或者多个
？：0个或者1个
{n}：n个
{n,m}：n个到m个
{n,}：最少n个
^：字符串的起始
$：字符串的结束

\d: 数字[0-9]
\w: 单词字符[a-zA-Z0-9_]
\s: 空白字符
\D: 非数字
\W: 非单词字符
\S: 非空白字符
