File用于表示文件系统中一个文件或目录
使用File可以：
	1.访问其表示的文件或者目录的属性信息
	2.创建，删除文件或目录
	3.访问一个目录的所有子项，但不能访问数据
File的创建需要指定路径：
	1.相对路径：不明确但是灵活，可以跨平台，推荐
	2.绝对路径：清晰明了但是不灵活，不利于跨平台
	./：表示当前目录，根据当前程序的运行环境而定
使用File创建文件：
	File file=new File("./xxx.txt");
	1.getName();获取文件名字
	2.length();获取文件大小
	3.boolean canRead();是否可读，返回布尔值
	   boolean canWriter();是否可写，返回布尔值
	   boolean isHidden();是否为隐藏文件，返回布尔值
	4.boolean isDirextory();判断是否为目录
	   boolean isFile();判断是否为文件
	   listFiles();获取该目录所有子项，以数组形式表示
	5.mkdir();新建一个目录
	   mkdirs();新建一个多级目录
